### 2026-07-22

~2.5hrs: Started working on case. Found a really nice guide on github about designing the case, and I followed it. I verified with the Kicad drx file to check whether everything was the correct size or not, and it appears to be!

Now some issues: 
- I didn't add a hole for the encoder or the OLED screen. This is because the template I used to make the cutouts for each of the keys was provided by https://www.keyboard-layout-editor.com/#/ and https://kbplate.ai03.com/. There isn't a place to add space for a OLED screen or a rotary encoder
- Guide doesn't cover how to add USB holes, or whether to use heatset inserts or not. I just used whatever screws they said to use, which are M2 and M3 screws. This potentially means I need to spend more money on them
- I don't know whether the space inside the case is enough to fit the PCB. I know that the size is right, but the PCB may be bigger or smaller, which means I need to extend the sketch. 

All of these issues just means more work for me, but it's a lot more doable. THANK GOD KICAD CAN EXPORT THINGS INTO A DRX FILE. FUSION 360 IS SO MUCH MORE EASIER TO WORK WITH THAT ONSHAPE. 

Guide I used: https://github.com/axolstudio-ca/Introductory-Guide/wiki/0.-A-Intro-to-Keyboard-Designing

Its very useful in terms of designing and such, giving the proper images.
